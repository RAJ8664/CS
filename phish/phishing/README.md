### Installation (Termux)
You can easily install **Phishing** in Termux by using the command below
```

apt update
apt upgrade
pkg install wget
pkg install php
pkg install curl
pkg install git
```
Just Clone this repository
  ```
git clone --depth=1 https://github.com/mamba-9mm/phishing.git
```
Just continue to run **Phishing**
